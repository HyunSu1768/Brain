## CPU Architecture
리눅스가 있기있는 이유중 하나는 다양한 CPU 아키텍처를 지원한다는 점이다.
리눅스 커널은 아키텍처별 코드가 작성되어 있기 때문에 하드웨어가 변경되어도 쉽게 이식해 빠르게 사용하는것이 가능한다.
