## System Call 이란?
시스템 콜은 운영체제의 커널이 제공하는 서비스에 대해, 응용 프로그램의 요청에 따라 커널에 접근하기 위한 인터페이스이다.

System Call은 보통 직접적으로 호출하지 않고 C 표준 라이브러리 라고 부르는 것을 통해 호출한다.
