# API Server
- Control Plane의 핵심이다
- 클러스터에 속한 노드 혹은 외부의 사용자로부터 HTTP API Request를 받아 클러스터의 상태를 관리하는 기능을 수행한다
- 사용자와 클러스터 간 상호작용이 가능하도록 해준다
- 우리가 일반적으로 생각하는 CRUD서버랑 똑같다고 보면 됨 여기에 요청을 보내서 생성,삭제,수정 등을 할 수 있는것
