## Docker(Container)의 단점
- 호스트 OS와의 성능 차이점이 발생할 수 있다. 컨테이너형 가상화를 제공하여 VM보다는 성능 손실이 적지만, 여전히 약간의 성능 손실은 존재한다.
- 기본적으로 컨테이너는 호스트 OS와 격리를 보장하지만, 완벽한 격리는 보장하지 못한다. 또한 호스트 OS에 문제가 생기면 컨테이너에도 장애가 전파된다.
