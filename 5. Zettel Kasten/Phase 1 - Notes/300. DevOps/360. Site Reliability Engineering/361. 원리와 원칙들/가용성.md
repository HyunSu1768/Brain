## 가용성
- 
가용성은 업타임을 기준으로 한 가용성과 성공한 요청수를 기준으로 한 가용성이 존재한다.

### 시간을 기준으로 한 가용성
```
가용성 = 업타임 / (업타임 + 다운타임 )
```

### 요청 수를 기준으로 한 가용성
```
가용성 = 성공한 요청 수 / 전체 요청 수
```
