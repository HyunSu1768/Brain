## RAM(Random Access Memory)

Ram은 프로그램이 실행되는 동안 필요한 정보를 저장하는 컴퓨터 메모리 이다.
프로세스가 주기억장치에 데이터를 저장하지 않는 이유는 속도차이 때문이다.
