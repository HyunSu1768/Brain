## 프로세스가 실행된다는 것은?
- 실행 가능한 코드가 메모리에 올라가 있고, 운영체제가 해당 코드에 CPU를 할당해서 코드가 실행될 수 있도록 만드는 것

### 프로그램 vs 프로세스
- 프로그램
	- 단순히 디스크에 존재하는 정적인 코드
		- chrome.exe, helloworld.c
- 프로세스
	- 코드가 메모리를 할당받고 실제 실행중인 동적 인스턴스
